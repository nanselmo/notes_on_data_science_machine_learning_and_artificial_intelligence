## Create Integer Variable


```scala
// Create an integer variable of 10
var i:Int = 10
```

## Increment And Decrement


```scala
// Increment up 1
i += 1

// View variable
i
```




    11




```scala
// Decrement up 1
i -= 1

// View variable
i
```




    10




```scala
// Increment up x2
i *= 2

// View variable
i
```




    20




```scala
// Decrement down by half
i /= 2

// View variable
i
```




    10
