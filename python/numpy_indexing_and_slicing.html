Indexing And Slicing Numpy Arrays¶

  • Author: Chris Albon, @ChrisAlbon
  • Date: -
  • Repo: Python 3 code snippets for data science
  • Note:

Import modules¶

In [2]:
import numpy as np

Create a 2x2 array¶

In [3]:
battle_deaths = [[344, 2345], [253, 4345]]
deaths = np.array(battle_deaths)
deaths
Out[3]:
array([[5341, 2345],
       [ 253, 4345]])

Select the top row, second item¶

In [4]:
deaths[0, 1]
Out[4]:
2345

Select the second column¶

In [6]:
deaths[:, 1]
Out[6]:
array([2345, 4345])

Select the second row¶

In [7]:
deaths[1, :]
Out[7]:
array([ 253, 4345])

Find the index of battles with less than 500 deaths¶

In [24]:
few_civ_deaths = np.where(civilian_deaths < 500)
few_civ_deaths
Out[24]:
(array([1, 3]),)

Find the number of civilian deaths in battles with less than 500 deaths¶

In [22]:
